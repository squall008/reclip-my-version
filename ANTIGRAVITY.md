# ANTIGRAVITY.md - ReClip プロジェクト憲法

## 1. プロジェクト概要
**ReClip** は、YouTube、TikTok、Instagram、Twitterなどの1000以上のサイトから動画や音声をダウンロードできる、クリーンなWeb UIを備えたオープンソースのセルフホスト型ツールです。

## 2. 目的・ターゲット
- **目的**: 煩わしい広告や制限なしに、お気に入りのコンテンツをオフライン用に保存する。
- **ターゲット**: 自分のサーバーやローカルPCでダウンローダーを管理したいユーザー。
- **利用シーン**: 動画制作の素材集め、オフライン視聴、音声抽出（MP3化）など。

## 3. 技術スタック
- **Backend**: Python 3.8+ / Flask
- **Frontend**: Vanilla HTML/CSS/JS (フレームワークなし、ビルド不要)
- **コアエンジン**: [yt-dlp](https://github.com/yt-dlp/yt-dlp)
- **変換ツール**: [FFmpeg](https://ffmpeg.org/)
- **デプロイ**: Docker対応

## 4. MVP（初期機能）
- [x] URL入力フォーム（複数一括対応）
- [x] 動画（MP4）/ 音声（MP3）の選択
- [x] 動画情報（タイトル、サムネイル、解像度）の取得
- [x] バックグラウンドでのダウンロード処理
- [x] ダウンロード完了後のファイル保存・提供
